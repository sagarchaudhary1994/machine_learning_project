# machine_learning_project

End to end machine learning project.

Requirement for this project-

1. Github Account
2. Heroku Account
3. VS Code IDE
4. Git CLI

Creating conda env

```
conda create -p venv python==3.7 -y
```

Activate conda env

```
conda activate venv/
```

Install requirements.txt

```
pip install -r requirements.txt
```
